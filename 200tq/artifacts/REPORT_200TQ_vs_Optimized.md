## 1. 전략 상세 설명

### 1.1 200TQ_Simple (TQQQ MA200 단순 스위칭)

```
규칙:
- TQQQ 종가 > MA(200) → 다음날 TQQQ 100%
- TQQQ 종가 ≤ MA(200) → 다음날 CASH(SHV) 100%
- 신호는 종가 확정 후 다음 거래일에 반영 (look-ahead bias 방지)
```

**장점:**

- 구현이 매우 단순함
- 큰 하락장에서 효과적으로 회피

**단점:**

- 횡보장에서 whipsaw 발생 가능
- 스탑로스/익절 없이 전략 일관성 유지

---

### 1.2 200TQ_SSOT_Full (아기티큐 SSOT 풀버전)

```
상태 판단:
- 하락장: TQQQ Close < MA(200) → 전량 CASH
- 집중 구간: MA(200) ≤ Close ≤ MA(200)*1.05 → TQQQ 매수
- 과열 구간: Close > MA(200)*1.05 → 신규 자금은 SPLG(S&P500)로

추가 규칙:
- 하루 더 확인: 하락→상승 전환 시 2일 연속 확인 후 진입
- 스탑로스: TQQQ 평균단가 대비 -5% 하락 시 전량 청산
- 부분 익절: +10%, +25%, +50%에서 각 10% 매도 → SPLG 재투자
- +100% 이상: 매 100%p마다 보유량의 50% 익절
```

**장점:**

- MDD 최저 (-50.72%)
- 변동성 최저 (30.36%)
- Sharpe 최고 (0.86)
- 리스크 관리 체계적

**단점:**

- 복잡한 규칙으로 구현/운영 부담
- 익절 규칙으로 인한 상승장 수익 일부 포기

---

### 1.3 Optim_QQQ_3_161 (QQQ MA 크로스 기반)

```
규칙:
- QQQ MA(3) > QQQ MA(161) → TQQQ 투자
- QQQ MA(3) ≤ QQQ MA(161) → CASH (또는 OFF 비중만큼 TQQQ 유지)

OFF 비중 변형:
- OFF0: 하락 신호 시 TQQQ 0%, CASH 100%
- OFF10: 하락 신호 시 TQQQ 10%, CASH 90%
- OFF20: 하락 신호 시 TQQQ 20%, CASH 80%
```

**핵심 차이점:**

- **TQQQ가 아닌 QQQ의 이평선** 사용 (기초자산 기준 판단)
- 더 짧은 MA(3)와 중기 MA(161) 크로스 사용
- OFF 비중으로 하락장에서도 일부 노출 유지 가능
